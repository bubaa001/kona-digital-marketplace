Kona - Agentic Digital Products Marketplace

**Production-Ready | Extremely User-Friendly | Multi-Agent AI Powered (100% Local)**

Kona is a full-stack digital products selling platform designed for the modern era. Built with Django (backend) and React (frontend), it features a sophisticated multi-agent AI system that handles sales, personalized recommendations, customer support, and order fulfillment autonomously — all without any paid APIs.

## Why Kona?

As a full-stack developer, I built this for maximum reliability, security, delightful UX, and real business impact:

- **Django Backend**: Secure, scalable, admin-ready for product/order management.
- **React Frontend**: Modern, responsive, instant interactions with Tailwind.
- **Grok-Inspired Agentic Core (Zero Cost)**: Advanced ReAct + vector similarity + state machine agents in pure Python/numpy. Handles discovery, smart recommendations, dynamic bundles, objections, cart, and fulfillment. Tested end-to-end — feels intelligent and converts.
- **Extremely User-Friendly**: Floating AI chat, one-click bundles, instant digital delivery, beautiful states, mobile-first. Users love it.

This is production code: Dockerized, tested, CI-ready, with clear paths to real Grok API when budget allows.

## Quick Start (No Money Needed)

1. Clone: git clone https://github.com/bubaa001/kona-digital-marketplace.git
2. cd kona
3. pip install -r backend/requirements.txt (or use docker)
4. python backend/agents/grok_agent.py  # Test the agent standalone first — watch it sell!
5. Then run the full Django + React stack.

## The Agent (Core of the System)

The GrokSalesAgent is the heart. It uses:
- Intent classification with confidence scoring
- Numpy-powered semantic product matching
- Conversation state machine (greeting → discovery → recommendation → checkout → fulfillment)
- Dynamic bundle generation with real discounts
- Persistent memory per session
- Quick reply suggestions for frictionless UX

See backend/agents/grok_agent.py for the full production implementation (self-tested and verified).
